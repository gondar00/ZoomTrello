Welcome to ZoomTrello!!!

BOARD

A user can Add a Card to each board as well as Clear the Board of all story cards.

CARDS

Once a card is added, it will be displayed on the board. The card is considered active if you hover on it. Edit the Title by hovering on the title text and clicking on the pencil icon. The same thing goes with the description.

You can also set the card color by clicking on the colored labels.

SEARCH

A user can Search a card by using the top right search filter.
